# School Transport Management System (STMS)

A desktop-based **School Transport Management System (STMS)** developed using **Python**, **PyQt6**, and **SQLite** to simplify school transport administration. The system enables administrators to efficiently manage students, parents, buses, drivers, and routes while providing parents with access to their child's transport information through a dedicated dashboard.


## Features

###  Authentication
- Secure Login System
- Admin Dashboard
- Parent Dashboard
- Role-Based Access

###  Student Management
- Add New Student
- Update Student Details
- Delete Student Records
- View Student Information
- Fee Management

###  Parent Management
- Add Parent Details
- Parent Login Credentials
- Update Parent Information
- Delete Parent Records

###  Bus Management
- Register New Buses
- Driver Assignment
- Route Assignment
- Bus Capacity Management

###  Route Management
- Create Routes
- Manage Pickup Points
- Assign Routes to Buses

###  Dashboard
- Total Students
- Active Buses
- Total Routes
- Quick Navigation Buttons

###  Database
- SQLite Database
- Relational Database Design
- CRUD Operations
- Parameterized SQL Queries

#  Installation

Clone the repository

```bash
git clone https://github.com/DivyanshuDubey10/School-Transport-Management-System.git
```

Go to the project directory

```bash
cd School-Transport-Management-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
